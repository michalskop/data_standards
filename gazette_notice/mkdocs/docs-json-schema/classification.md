## Klasifikace <small>(Classification)</small>

Klasifikace ze seznamu (číselníku, apod.)

**`classification.json`**
#### code

Alternativy: `code`

Kód, klasifikace

Typ: `string`

#### name

Alternativy: `name`

Název pro danou klasifikaci, zvláště pokud hodnota je köd

Typ: `string`

#### scheme

Alternativy: `scheme`

Schéma klasifikace, např. 'ruian/budovy'

Typ: `string`

### Příklad


```json
{
    "identifier": "4132908",
    "scheme": "RÚIAN/budovy"
}
```
